# docs 资料说明

本目录存放族谱相关原始资料与提取文本，用于完善本站屈氏信息及屈氏人员数据。

## 文件列表

| 文件 | 说明 |
|------|------|
| **屈氏族谱（陕西谓南屈仲辉提供）.doc** | 原始 Word 文档，由陕西渭南屈仲辉提供，含屈氏起源、迁徙分布、字辈排行、郡望堂号、名人等。 |
| **屈氏族谱（陕西谓南屈仲辉提供）.txt** | 由脚本从上述 .doc 提取的 UTF-8 文本（运行 `python -m scripts.extract_doc_text` 生成），供入库与屈氏文化页参考。 |
| **屈氏起源_名人_族谱（陕西谓南屈仲辉提供）.pdf** | PDF 版屈氏文化资料。 |
| **直系宗亲简图（贵州余庆屈氏）01.pdf** | 贵州余庆屈氏直系宗亲简图。 |
| **湘乡屈氏系图.pdf** | 湘乡屈氏系图。 |

## 使用方式

1. **提取 .doc 文本**（需安装 `olefile`）  
   在项目根目录执行：  
   `python -m scripts.extract_doc_text`  
   会生成同名的 `.txt` 文件，便于检索与引用。

2. **根据 docs 完善屈氏信息与人员**  
   执行：  
   `python -m scripts.import_from_docs`  
   - 若尚未存在，会创建族谱「陕西渭南屈氏族谱（屈仲辉提供）」；  
   - 并添加成员「屈仲辉」（资料提供者）。  

3. 本站「屈氏文化」页中的**名人举例**、**各地字辈摘录**等已据上述 txt 内容整理展示。

## 备注

- 文件名中「谓南」为「渭南」之误写，内容指陕西渭南。
- 新增或更新 .doc/.pdf 后，可重新运行提取脚本更新 .txt；人员与族谱需在系统中通过「族谱库」「成员管理」或脚本维护。
